This tool displays the assets from Arcanox.apk, which is available from Google Play, although the game no longer works. This is just a tool to graphically show assets and names. It shows animations, but is glitchy. Bottom right whitespace is ommitted, causing some animations to resize, (and others would resize anyway,) making the page jump around a bit. This tool follows atlas file organization, though the game did not always. This animates different versions of some assets and shows some animations as different versions, because that's how they are organized in the apk, though that's not how the game handled them.
